## 화면 비율 설정
**프리미어 프로 상단바** 
```
FILE > NEW > SEQUENCE > Settings > Frame size, horizontal 조정
```
제미나이 생성 이미지 사이즈 :  1024 x 1024

**15:8** 비율 변경시 세로 길이
> **15:8 = 1024 : 546.13**

**1:1** 비율 변경시 세로 길이
> **1:1 = 1024 : 1024**

## 영상 제작
```
1. 화면 오른쪽 상단 **Quick Export 로고 클릭**
2. 파일 경로 설정
3. export
```
